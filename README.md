# Strong Coding Skill Roadmap

이 레포지토리는 **2026년 2월 7일 ~ 2026년 6월 중순**까지  
Strong Coding Skill 갖추기를 목표로 하는 “Strong Coding Skill” 학습 과정을 기록하는 공간입니다.[web:6][web:9][web:22]

## 🎯 목표

- 알고리즘/자료구조를 활용해 중급 난이도 문제를 스스로 해결할 수 있는 실력 달성
- Python 중심으로, 필요시 C++까지 활용 가능한 코딩 역량 확보
- ML·Computer Vision 기초를 익히고, 작은 프로젝트 1–2개 완성
- 깃허브에 성장과 결과물을 꾸준히 남겨 보여줄 수 있는 실력 만들기

## ⏰ 기간

- 전체 기간: 2026-02-07 ~ 2026-06-15 (이후에도 간간히 작성 예정)
- 개강일: 2026-03-03 (학업·팀 프로젝트와 병행)
- 주당 투자 시간
  - 개강 전(2/7 ~ 3/2): 하루 5~7시간(집중 학습)
  - 학기 중(3/3 ~ 6/15): 평일 2~3시간, 주말 4~6시간

## 📌 주차별 큰 계획

> 상세한 일정과 공부 내용은 `/plans` 디렉토리와 커밋 메시지로 관리합니다.

### 1️⃣ 2/7 ~ 3/2: 언어 + 알고리즘 기초 다지기

- Python(or C++) 문법 및 기본 문법 정리
- 기본 자료구조: 배열, 리스트, 스택, 큐, 해시, 문자열
- 알고리즘 기초: 완전탐색, 정렬, 재귀, DFS/BFS, 이진탐색, 기초 DP
- 백준/프로그래머스 문제를 통한 브론즈~실버 난이도 문제풀이 루틴 만들기

### 2️⃣ 3/3 ~ 3/31: 학기 적응 + 알고리즘 루틴 고정

- 평일: 알고리즘 1~1.5시간(문제 2~4개), ML/Python 복습 0.5~1시간
- 주말: 알고리즘 심화 + 작은 ML 실습
- 목표: 실버 전반 안정화, 골드 입구 난이도 도전

### 3️⃣ 4월: 알고리즘 심화 + 딥러닝 입문

- 알고리즘: 그래프, DP, 그리디, 구현을 균형 있게 연습
- 딥러닝 기초: PyTorch(or TensorFlow) 설치 및 기본 모델 구현
- MNIST/CIFAR-10 같은 데이터셋으로 간단한 CNN 학습 실습

### 4️⃣ 5월: ML·CV 관련 작은 프로젝트 시작

- 예시 프로젝트
  - 이미지 분류 성능 개선 실험
  - 간단한 포즈 추정/객체 검출 튜토리얼 응용
- 목표
  - “문제 정의 → 접근 방법 → 실험 → 결과 → 한계/개선점”이 있는 프로젝트 1개 이상 완성

### 5️⃣ 6월 초 ~ 중순: 마무리 및 포트폴리오 정리

- 알고리즘: 골드 중급 문제 도전 + 약점 보완
- 프로젝트: README 정리, 결과 정리, 코드 리팩토링
- 전체 기간 회고: 무엇을 배웠는지, 어떤 역량이 생겼는지 텍스트로 정리
